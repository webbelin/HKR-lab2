Biography

My name is Elin and I live in the southern part of Sweden. I've worked as a webdesigner and web editor for the last ten years, mainly focusing on animal welfare and environmental sustainability issues.

Around 15 years ago I studied media technology at Södertörns University. After that I became involved in webproduction of different kinds, both as a web editor but also as a Wordpress webdesigner. I've been learning Wordpress, html and css along the way but have never managed to find the time to deepen my knowledge in related technologies and coding languages, for example Javascript.

I'm taking this course because I want to start learning Javascript and close some gaps in my knowledge of Html and CSS but also to be able to graduate with a Bachelor's Degree in Media technology - since I only miss one course to do that.

Hobbies
